# How to Install Wise Family Expenses as a Mobile App

This app is a Progressive Web App (PWA) that can be installed on your phone and used like a native app.

## üì± Installation Instructions

**Your app is deployed at**: `https://wise-family-expenses.vercel.app`

### For Android Phones (Chrome/Edge)

#### Method 1: Automatic Prompt (Recommended)
1. **Open Chrome or Edge** on your Android phone
2. **Go to**: `https://wise-family-expenses.vercel.app`
3. **Sign in** with Google (if not already signed in)
4. **Look for the install banner** at the bottom that says "Add to Home screen" or "Install app"
5. **Tap "Install"** or "Add"
6. **Done!** The app icon will appear on your home screen

#### Method 2: Manual (If prompt doesn't show)
1. **Open Chrome or Edge** on your Android phone
2. **Go to**: `https://wise-family-expenses.vercel.app`
3. **Sign in** with Google (if not already signed in)
4. **Tap the menu** (three dots ‚ò∞) in the top-right corner
5. **Tap "Add to Home screen"** or **"Install app"**
6. **Tap "Install"** or "Add"
7. **Done!** The app icon will appear on your home screen

### For iPhone/iPad (Safari)

1. **Open Safari** on your iPhone/iPad
2. **Go to**: `https://wise-family-expenses.vercel.app`
3. **Sign in** with Google (if not already signed in)
4. **Tap the Share button** (square with arrow pointing up) at the bottom
5. **Scroll down** in the share menu and tap **"Add to Home Screen"**
6. **Customize the name** (optional) - you can change it before adding
7. **Tap "Add"** in the top-right corner
8. **Done!** The app icon will appear on your home screen

## üöÄ Deploying for Installation

### Option 1: Deploy to Vercel (Recommended)

1. **Push your code to GitHub**
   ```bash
   git add .
   git commit -m "Ready for deployment"
   git push
   ```

2. **Deploy to Vercel**
   - Go to [vercel.com](https://vercel.com)
   - Import your GitHub repository
   - **Important:** Initialize database first by running `npm run db:init` locally
   - Add environment variables:
     - `DATABASE_URL` (your Neon read-write connection string)
     - **Do NOT add `DATABASE_URL_DDL`** - Only needed for development/setup
   - Deploy!

3. **Access the deployed URL**
   - Vercel will give you a URL like `https://your-app.vercel.app`
   - Open this URL on your phone to install

### Option 2: Deploy to Other Platforms

- **Netlify**: Similar to Vercel, supports Next.js and PWAs
- **Railway**: Good for full-stack apps
- **Your own server**: Make sure HTTPS is enabled

## üîß Development vs Production

- **Development (`npm run dev`)**: PWA features are disabled to avoid build issues
- **Production (`npm run build && npm start`)**: PWA features are fully enabled

### Testing PWA Features Locally

1. **Build the app**
   ```bash
   npm run build
   npm start
   ```

2. **Access via your local network**
   - Find your computer's IP address
   - Access via `http://YOUR_IP:3000` from your phone
   - Note: Some PWA features require HTTPS even in development

3. **Use ngrok for HTTPS testing** (optional)
   ```bash
   npx ngrok http 3000
   ```
   - Use the ngrok HTTPS URL on your phone

## üé® Customizing the App Icon

To change the app icon:

1. Replace `public/icon-192.png` (192x192 pixels)
2. Replace `public/icon-512.png` (512x512 pixels)
3. Replace `public/favicon.ico` (for browser tab)
4. Rebuild the app: `npm run build`

## üìù Notes

- The app works offline after first visit (cached by service worker)
- Data syncs when you're back online
- App updates automatically when you deploy new versions
- The app icon and name are configured in `public/manifest.json`

## üêõ Troubleshooting

**"Add to Home Screen" option not showing?**
- Make sure you're using the deployed HTTPS version
- Clear browser cache and try again
- Check that manifest.json is accessible at `/manifest.json`

**App not installing?**
- Ensure you're on HTTPS (not HTTP, except localhost)
- Check browser console for errors
- Verify manifest.json is valid JSON

**App icon not showing?**
- Check that icon files exist in `public/` folder
- Verify icon paths in manifest.json
- Clear browser cache
